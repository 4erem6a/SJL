# Sample JVM Language
Sample JVM JIT compiler for a sample language created for educational purposes.
# A few examples:
```
# Sample "Hello, world!" program:
print char 'H'; 'e'; 'l'; 'l'; 'o'; ','; ' '; 'w'; 'o'; 'r'; 'l'; 'd'; '!'; 10
```
```
# Some math & variables:
$x = 10
$y = 3.14
println $x * ($y + ($x / -$y))
```
```
# Some user input:
$i = input
print $i
print char ' '; '*'; ' '
print $i
print char ' '; '='; ' '
println $i * $i
```
